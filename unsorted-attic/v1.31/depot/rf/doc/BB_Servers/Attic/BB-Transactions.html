BB Transactions

The following lists the transactions initiated by BB Servers. The details of the transactions are given in separate documents and indicates interactions between the different servers, their effects on the data stores and the realtime/synchronous or deferred nature. 

  1. BB Depot - System Monitor
    • Register BB Depot
    • Report BB Depot Status
  2. BB Depot - Software Update
    • BB Depot Software Update Check
  3. BB Depot - Content Manager
    • Content Synchronization
  4. BB Depot - Retail App
    • BB Activation
    • BB Deactivation
    • Purchase eTicket
    • Refund eTicket
    • Purchase hardware options
  5. BB Depot - Inventory App
    • Order Inventory (includes BB, accessories)
    • Receive Inventory
  6. BB Depot - Installer App
    • Register BB Depot
    • Register DepotID
    • Register Download Station
  7. BB Depot - BB Recovery Manager
    • Reinitialize BB Card
  8. Download Station - Download
    • Download Content to BB Card
  9. Download Station - eTicket Manager
    • Browse eTicket Info
    • Download/Recover eTicket
  10. BB Player - Re-Init Server
    • This server is passive and does not initiate transactions, but participates in the initialization of the memory card.
  11. Transaction - Content Availability Server
    • Start Content Distribution
    • Revoke Content
  12. Transaction - BB Activation Server, eTicket Server, Inventory Server, Installer Server, BB Recovery Server
    • These servers are passive and do  not initiate transactions
  13. Manufacturer - BB Chip Manufacture
    • BB Chip Manufacture - when chips are manufactured & BB private key is burnt into the chip
  14. Manufacturer - BB Shipment Manager
    • BB Shipment- when completed unit is manufactured and shipped from the factory
  15. Release Kit  - Content Packager 
    • Package Content - when content is developed and ready to be send to the studio server
  16. Release Kit - Content Release Manager
    • Release Content - release content to the studio server
  17. Content Management - Release Kit Manager
    • Register Studio - create new studio
  18. Content Management - Validation Server
    • Receive Content - receives new content from studio Release Kit
    • Validate Content
    • Package Content
  19. Content Management - Publishing Server
    • Publish Content - pushes content & meta data to the CDS & XS
  20. Content Distribution
    • This server is passive and does not initiate transactions, but participates in Publish Content, Content Synchronization & Download Content to BB Depot transactions.
  21. RMS - Content Notifier
    • Notify Depot
  22. RMS - System Status, Statistics, Software Update
    • These servers are passive and do not initiate transactions.
  23. Certificate Authority - BB Key Manager
    • Generate BBID
    • Update BB
  24. Certificate Authority - BB CA
    • Issue BB DC
  25. Certificate Authority - License Manager CA
    • Issue License Manager DC
    • Revoke License Manager DC
  26. Certificate Authority - Studio CA
    • Issue Studio DC - when current certificate expires or is compromised
  27. Certificate Authority - Depot CA
  28. Certificate Authority - Server CA
  29. BB Software - Software Release Manager
    • Release BB Software
  30. Customer Service
    • BB
      • Select Free Game - for use with BB Activation
      • Return Defective BB Player - credit retailer
      • Revoke eTicket - if customer has reported problems with ContentID
      • Issue eTicket - issue a replacement for a revoked eTicket
    • Region
      • Create Region
    • Retailer/Store
      • Create Retailer
      • Activate Retailer
      • Suspend Retailer
      • Create Store
      • Open Store
      • Close Store
    • Depot
      • Create BB Depot
      • Suspend BB Depot
      • Activate DepotID
      • Suspend DepotID
    • Inventory
      • Place New Order
      • Cancel Order/Modify Order
    • Network Failure Recovery
      • Execute 'eTicket Refund' on behalf of retailer
      • Execute 'BB Player Return' on behalf of retailer
  31. Billing/Account Management
