BB Transactions

The following lists the transactions initiated by BB Servers. The details of the transactions are given in separate documents and indicates interactions between the different servers, their effects on the data stores and the realtime/synchronous or deferred nature.

  1. BB Depot - System Monitor
    • Report BB Depot Status (RMS)
  2. BB Depot - Software Update
    • BB Depot Software Update Check
  3. BB Depot - Content Manager
    • Content Synchronization
  4. BB Depot - Retail App
    • Retail App startup validation:
      • match MAC address against HardwareID (local to Depot)
      • match DepotID against HardwareID
    • BB Activation Replaced by Factory BB Activation
    • BB Deactivation
    • Purchase eTicket
    • Refund eTicket
  5. BB Depot - Installer App
    • Configure BB Depot
    • Register HardwareID
  6. BB Depot - BB Recovery Manager
    • Reinitialize BB Card
  7. Download Station - Download
    • Download Content to BB Card
  8. Download Station - eTicket Manager
    • Browse eTicket Info
    • Recover eTicket
  9. BB Player - Re-Init Server
    • This server is passive and does not initiate transactions, but participates in the initialization of the memory card.
  10. Transaction - Content Availability Server
    • Start Content Distribution
    • Revoke Content
  11. Transaction - BB Activation Server, eTicket Server, Inventory Server, Installer Server, BB Recovery Server
    • These servers are passive and do  not initiate transactions
  12. Manufacturer - BB Chip Manufacture
    • BB Chip manufactured - when chips are manufactured & BB private key is burnt into the chip
  13. Manufacturer - BB Shipment Manager
    • BB Shipment- when completed BB Player is manufactured and shipped from the factory
  14. Release Kit  - Content Packager 
    • Package Content - when content is developed and ready to be send to the studio server
  15. Release Kit - Content Release Manager
    • Release Content - release content to the studio server
  16. Content Management - Release Kit Manager
    • Register Studio - create new studio
  17. Content Management - Validation Server
    • Receive Content - receives new content from studio Release Kit
    • Validate Content
    • Package Content
  18. Content Management - Publishing Server
    • Publish Content - pushes content & meta data to the CDS & XS
  19. Content Distribution
    • This server is passive and does not initiate transactions, but participates in Publish Content, Content Synchronization & Download Content to BB Depot transactions.
  20. RMS - Content Notifier
    • Notify Depot
  21. RMS - System Status, Statistics, Software Update
    • These servers are passive and do not initiate transactions.
  22. Certificate Authority - BB Key Manager
    • Register BB - available for customer purchase
  23. Certificate Authority
    • Issue License Manager DC
    • Revoke License Manager DC
    • Issue BB Depot DepotID
    • Issue BB Depot HardwareID
  24. BB Software - Software Release Manager
    • Release BB Software
  25. Customer Service
    • BB
      • Select Free Game - for use with BB Activation
      • Replace BB - defective BB Player is replaced by alternate, eTickets are reissued
      • Suspend BB Player - does not function any more or is suspended due to security reasons 
    • Region
      • Create Region
    • Retailer/Store
      • Create Retailer
      • Activate Retailer
      • Suspend Retailer
      • Create Store
      • Open Store
      • Close Store
      • Activate DepotID
      • Suspend DepotID
    • Depot
      • Create BB Depot
      • Suspend BB Depot
    • Inventory
      • Place New Order
      • Cancel Order/Modify Order
      • Delivered Order
  26. Billing/Account Management
