Control Flow - users, processes, servers & their actions

  1. User/Client/Process
    1. Action - Server

 

  1. Manufacturer

  2. BB Depot - Software Update

    • Download new software - RMS
  3. BB Depot - System Monitor
    • Register with RMS - RMS
    • Software Status report - RMS
  4. BB Depot - Content Manager
    • Content Synchronization - CDS
    • Content Download - CDS
  5. BB Depot - Retail Application/Store Owner
    • BB Activation - BB Activation Server, BB Inventory Server

    • Place order for BBs & hardware options - BB Inventory Server
    • Receive inventory - BB Inventory Server
  7. BB Depot - Install Manager
    • Register BB Depot - Depot Account Transaction Server
    • Register Tokens - Depot Account Transaction Server
    • Register Download Stations - Depot Account Transaction Server
  8. BB Player - Browser Application
    • Query list of available applications - Depot Content Manager
  9. BB Player - Content Download Application
    • Download application into BB  - Depot Content Manager
  10. BB Player - eTicket Manager
    • Confirm eTicket purchase - eTicket Transaction Server
  11. BB Player - Recovery Manager
    • Memory card initialization application - BB Activation Server
    • eTicket Recovery - eTicket Transaction Server
  12. Studio - Content Development Manager

  17. Content Availability Server

  18. BB Software Release Manager
