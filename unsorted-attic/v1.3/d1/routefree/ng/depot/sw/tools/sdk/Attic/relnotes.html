IOP SDK Release Notes

$Revision: 1.2 $
$Date: 2006/04/04 05:51:15 $

Summary

This release (version 1.1.0) contains the following features/bug fixes:
  • Increase number of system resources for IOS development such as:
    • Threads: 32
    • Message Queues: 128
    • Timers: 64
    • File descriptors: 64

  • Include SDIO register definitions in $ROOT/usr/src/signum/rvl_iop.def


  • Include new symbol definitions and address map for WD and WL in $ROOT/usr/include/make/NGaddrs_debug:
    • MK_WD_START
    • MK_WL_START
    These above definitions should be used in WD and WL Makefile to ensure that correct address space is used for each module.


  • Added the following functions to libioslibc.a lilbrary:
    • strncat()
    • snprintf()
    • vsnprintf()

  • Fix IOS_FlushDCache() to correctly handle number of cache lines to flush.


  • Fix IOS_FlushMem() by providing critical section protection when accessing hardware registers.

  • Fix IOS_Alloc() to handle back link correctly.
When launched, the iossh program should display: 
     [build time: 04/03/06 08:25:34 PM  OS version: 00005000]

Known Issues

  • To run any of the test programs properly, make sure that you type "exit" in the the VIO window (although a prompt doesn't appear) after you type "run".
