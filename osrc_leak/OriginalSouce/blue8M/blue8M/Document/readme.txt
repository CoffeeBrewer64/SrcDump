●日本語スクリプトの説明


用語の定義
	
	下図はメッセージウィンドウの図です。
	
	+-----------------------------+
	|                             |
	|Are you Keiko?               |
	|                             |
	|How've you been doing?       |
	+-----------------------------+
	
	メッセージウィンドウ内の位置について以下の用語を定義します。
	
	トップポジション → 上図 "Ａｒｅ"の"Ａ"の位置
	ホームポジション → 上図 "Ｈｏｗ’"の"Ｈ"の位置




{cr@}
	カーソルを下行、書き出し列へ移動します。
	
	例１：
	
	+-------------------------+
	|                         |
	|こんにちは               |
	|                         |
	|ようこそ！               |
	+-------------------------+

		こんにちは{cr@}
		ようこそ！

	例１の場合は見た目は次の{home@}と同じになります。


	例２：

	+-------------------------+
	|                         |
	|      こんにちは         |
	|                         |
	|      ようこそ！         |
	+-------------------------+

		こんにちは{cr@}
		ようこそ！

	例２の場合「こんにちは」の書き出し位置に合わせて「ようこそ！」
	を表示。



{home@}
	これはカーソルをホームポジションへ移動させます。
	
	+-------------------------+
	|                         |
	|こんにちは               |
	|                         |
	|ようこそ！               |
	+-------------------------+
	
	上の図の様にメッセージを出すには、
	
		こんにちは{home@}
		ようこそ！
	
	とします。


{scr@}
	メッセージウィンドウ内をスクロールさせます。
	プロンプト "▼"表示やＡＢボタン待ちがないことを除けば見た目は
	次の{gyo3@}と同じです。


{scrwait@}
	動作は次の{gyo3@}と同じです。
	

{gyo3@}
	これはプレーヤにＡボタンを押させるためのプロンプト "▼"を
	点滅表示し、プレーヤがＡ又はＢボタンを押したら１行上に
	スクロールします。
	
	例えば、
	
	+------------------------------------+
	|                                    |
	|はじめまして！                      |
	|                                    |
	|ポケット  モンスターの  せかいへ  ▼|
	+------------------------------------+
	
	これをスクリプトで書くと、
	
		はじめまして！{home@}
		ポケット  モンスターの  せかいへ{gyou3@}
	
	となります。ここでプレーヤがボタンを押すと、
	
	+------------------------------------+
	|                                    |
	|ポケット  モンスターの  せかいへ    |
	|                                    |
	|                                    |
	+------------------------------------+
	
	と表示されます。この時次のメッセージを書き出す位置は
	ホームポジションです。


{normal@}
	メッセージウィンドウを表示させた時の見た目は {gyo3@}と
	似てますが少し機能が違います。
	
	+------------------------------------+
	|                                    |
	|ポケット  モンスターの  せかいへ    |
	|                                    |
	|ようこそ！                        ▼|
	+------------------------------------+
	
	ここまでは gyo3と同じでＡＢボタン待ちに入ります。
	ここでプレーヤがボタンを押すと、
	
	+------------------------------------+
	|                                    |
	|                                    |
	|                                    |
	|                                    |
	+------------------------------------+
	
	メッセージウィンドウ内部をクリアして、
	次のメッセージの書き出し位置をトップポジションに設定します。


{EOM^1}, {EOM^2}, {EOMeom}, {EOMwaiteom}
	これらはメッセージの終了コードです。１つのメッセージの最後
	に必ず入ります。
	
	{EOM^1}, {EOM^2}, {EOMeom} は単純にメッセージはここまでだ！とピリオドの役割
	をしているだけですが、{EOMwaiteom} は追加機能があります。
	
	{EOMwaiteom} は見た目は {normal@} に似ています。
	
	+------------------------------------+
	|                                    |
	|みんなからは  ポケモン  はかせと    |
	|                                    |
	|したわれて  おるよ                ▼|
	+------------------------------------+
	
	これでＡＢボタン待ちに入ります。
	ここでプレーヤがＡボタンを押すと、"▼"を消して処理を終了します。



以下の文字列は置き換えられます。


{myname@}
	これはゲームの最初に登録した自分の名前に置き換えられます。


{rival@}
	これはゲームの最初に登録したライバルの名前に置き換えられます。


[変数名]
	これは持ち物名、ポケモン名等に置き換えられます。
	
	例：
		ぼく　おぼえてます？{home@}
		ほら……！{gyo3@}
		オーキド　はかせの　じょしゅ　です{normal@}
		
		ポケモンずかんが　(DA_BUF + 0)しゅるい{home@}
		あつまってる　ようなら{gyo3@}
		[anime_buf + 0]を　わたす　よう{gyo3@}
		はかせに　いわれ　ました{normal@}
		
		……　それでは　さっそく{home@}
		{myname}くん！{gyo3@}
		つかまえた　ポケモンは{gyo3@}
		(DA_BUF + 0)しゅるい　より　おおい　かな？{EOMeom}


(変数名)
	これは数字等に置き換えられます。上述の例を参照。


<変数名>
	これも数字等に置き換えられます。
	
	例：
		{myname}は　しょうきんとして{home@}
		<yuuichi_gold>{yen@}　てにいれた！{EOMwaiteom}



●英語への翻訳規則
  (1) 使用する文字は全て半角文字を使用して下さい。
  (2) １つのメッセージは全て
      
        message_id:XXXXXXXXXX
        ....................
        ...................{終了コード}
      
      この間に記述します。

  (3) 行の先頭にある "//" はコメント行です。これはプログラムに翻訳されません。
      予め日本語メッセージには、その台詞がどこで語られているかを分かりやすくする
      ためにコメント行にて補足が加えてあります。（全てではありません）翻訳のさい
      の参考にして下さい。

  (4) 翻訳具体例
  
// 日本語
message_id:msg_fossil_get_2_D02R0103
{myname}は{home@}
[str_buf]を　てにいれた！{EOM^2}


// 英語
message_id:msg_fossil_get_2_D02R0103	← この行は絶対に編集しないで下さい！
{myname}{home@}
got a [str_buf]！{EOM^2}


    {....},[....],<.....>,(....)等はそのままお使い下さい。括弧の中を編集しては
    いけません。
    

    もし特殊文字を表示させたい場合はご相談下さい。例えば日本語では、

message_id:safari_end_msg1_23_EV_TOOL
アナウンス『ピンポーン！{normal@}

     の様に "『" を使用してますが、英語の場合他の文字にしたいとか・・・。
